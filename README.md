MatosDev Portfolio
Portafolio personal de MatosDev: sitio web moderno para presentar servicios, proyectos y formas de contacto.

Stack
React (CRA/CRACO)

Tailwind CSS

React Router

React Icons
Requisitos
Node.js + npm

Instalación
bash
cd frontend
npm install
Desarrollo (modo local)
bash
cd frontend
npm start
Abre http://localhost:3000.

Build (producción)
bash
cd frontend
npm run build
Estructura
frontend/: aplicación web (UI)

backend/: (si aplica) servicios/recursos del lado servidor
